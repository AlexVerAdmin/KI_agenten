# Архитектура проекта "Antigravity Agents" (Hybrid Cloud & DevOps)

Этот документ описывает текущую конфигурацию сети, принципы работы агентов и стратегию безопасности для использования в последующих сессиях.

---

## 🏗 1. Инфраструктура и Сеть (Hybrid Infrastructure)

### Узел 1: Внешний сервер (VDS)
- **Роль**: Постоянная точка входа (24/7), VPN-хост (WireGuard Server).
- **Стек**: Docker, Traefik, Authelia (защита через `secured-chain@file`).
- **Компоненты**:
    *   **Telegram-бот**: Основной процесс обработки сообщений.
    *   **Web UI (Streamlit)**: Доступ по адресу `https://agents.eurostick.com.ua`.
    *   **Syncthing**: Синхронизация папки `/app/obsidian` с локальным сервером.
- **Сеть**: Внешний IP для доменов + Внутренний IP в туннеле WG (`10.0.0.1`).

### Узел 2: Локальный сервер (Proxmox Home Lab)
- **Роль**: Мощный вычислительный узел (Heavy Lifting).
- **Железо**: NVIDIA Tesla P4000 (8GB VRAM).
- **Стек**: Docker (LXC/Bare Metal) с NVIDIA Container Toolkit.
- **VPN**: Подключен к VDS как клиент через WireGuard (настроен на OPNsense).
- **Компоненты**:
    *   **Ollama (GPU)**: Локальный запуск моделей (Llama 3.1 8B, Mistral).
    *   **ChromaDB**: Векторная база данных для RAG (Obsidian).
- **Сеть**: Локальный IP в домашней сети (`например, 192.168.x.x`) + IP в туннеле WG (`например, 10.0.0.2`).

## 🤖 2. Логика Агентов и Выбор Моделей (Routing Logic)

### Группы Агентов:
1.  **Учитель (Herr Max Klein) & Консультант (HR)**:
    - **Приоритет**: Google Gemini 2.5 Pro/Flash.
    - **API**: `v1beta`.
    - **Причина**: Огромный контекст (128k+) и высокая скорость для обучения.
2.  **Оркестратор (Supervisor)**:
    - **Приоритет**: Локальная модель (Ollama) -> Groq (Llama-3.3-70b) как Backup.
    - **Причина**: Безопасность планирования и конфиденциальность запросов.
3.  **Системные Агенты (DevOps)**:
    - `vds_admin`: Мониторинг логов Docker, Traefik, статуса VPN на VDS.
    - `local_admin`: Мониторинг GPU P4000, температуры, процессов на Proxmox.

## 🔒 3. Протокол Безопасности и Контроля (DevOps Control)

### Human-in-the-Loop (HITL)
- **Терминал**: Агенты-админы имеют доступ к выполнению команд, но **только под ручным подтверждением**.
- **Процесс**: 
    1. Агент предлагает команду в чате Streamlit.
    2. Появляется кнопка «Разрешить исполнение».
    3. Команда выполняется только после клика пользователя.
- **Изоляция**: Инструменты `vds_admin` и `local_admin` разделены. Ошибка настройки локального сервера не должна влиять на доступность VDS.
