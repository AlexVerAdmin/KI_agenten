# Antigravity Agents 🚀 — Ваша персональная ИИ-экосистема

**Antigravity Agents** — это не просто чат-бот, а масштабируемая платформа для создания и управления многоагентными ИИ-системами. Проект объединяет мощь современных LLM (Gemini, Groq, Llama, Mistral) с вашими личными данными (Obsidian, документы) в единый интеллектуальный хаб.

---

## 🔥 Ключевые возможности

### 1. Многоагентная Интеллектуальность (Multi-Agent RAG)
Система построена на базе **LangGraph**, что позволяет создавать сложные сценарии взаимодействия.
- **Специализированные агенты**: Каждый агент (General, JobSearch, Research) имеет свои инструкции, личность и доступ к разным инструментам.
- **Память (Short-term & Long-term)**: Использование SQLite для истории диалогов и векторной базы Chroma для хранения знаний.

### 2. Интеграция "Второго мозга"
Прямое подключение к вашим локальным хранилищам данных:
- **Obsidian**: Агенты могут использовать ваши заметки для контекста, отвечая на вопросы по вашей личной базе знаний.
- **JobSearch**: Специальный агент для работы с резюме, описаниями вакансий и материалами для подготовки к интервью.

### 3. Мультимодальность и Интерфейсы
- **Голосовое управление**: Telegram-бот понимает голосовые сообщения (через **Whisper**) и отвечает естественным голосом (**Edge TTS**).
- **Streamlit UI**: Профессиональная веб-панель для управления агентами и мониторинга диалогов в реальном времени.

### 4. Конфиденциальность и Гибкость
- **Локальные модели**: Поддержка Ollama позволяет запускать ИИ полностью на вашем железе.
- **Разделение доступа**: Возможность ограничить круг пользователей Telegram по `USER_IDS`.

---

## 🏗 Архитектура и Масштабирование

Проект спроектирован с учетом легкого расширения:
- **Добавление новых агентов**: Достаточно описать нового агента в `AGENT_REGISTRY` в [core/orchestrator.py](core/orchestrator.py).
- **Интеграция сторонних API**: Готовая структура для подключения новых LLM-провайдеров.
- **Масштабирование данных**: Векторное хранилище Chroma позволяет индексировать тысячи документов без потери производительности.

---

## 💎 Преимущества

1. **Скорость**: Использование Groq API обеспечивает практически мгновенную генерацию текста.
2. **Контроль данных**: Ваши документы для поиска работы и заметки Obsidian остаются на вашем диске.
3. **Автономность**: Система готова к работе в офлайн-режиме (при использовании Ollama).
4. **Удобство**: Веб-интерфейс для глубокой работы на ПК и Telegram-бот для быстрого доступа с телефона.

## 🛠 Установка и Настройка

### Предварительные условия
- Python 3.10+
- (Опционально) FFmpeg (для работы с аудио в Telegram)

### Шаги запуска
1. **Клонирование и зависимости**:
   ```bash
   git clone https://github.com/your-username/AntigravityAgents.git
   cd AntigravityAgents
   pip install -r requirements.txt
   ```

2. **Настройка окружения**:
   Создайте `.env` на базе шаблона и заполните ключи:
   ```bash
   cp .env.example .env
   ```

3. **Запустите проект**:
   - Веб-интерфейс: `run_ui.bat`
   - Telegram-бот: `run_bot.bat`

Подробное описание архитектуры и файлов проекта можно найти в [project_structure.md](project_structure.md).
